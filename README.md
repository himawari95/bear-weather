
# 小熊天气AI助手

这是一个结合天气查询与AI对话功能的Web项目，用户可以通过自然语言和AI聊天，获取实时天气信息。

---

## 项目功能
- 实时天气查询：获取当前城市的温度、湿度、风力等数据
- AI对话交互：支持口语化提问，AI会以可爱的语气回复
- 上下文对话：记住用户提到的城市和问题，连贯回答

---

## 运行方式
1. 克隆项目到本地
```bash
git clone git@github.com:himawari95/bear-weather.git
cd bear-weather
```

2. 启动后端服务
```bash
node server.js
```

3. 打开浏览器访问
```
http://localhost:3000/weather_website.html
```

---

## 技术栈
- 前端：HTML / CSS / JavaScript
- 后端：Node.js + Express
- AI模型：vivo蓝心大模型
- 天气数据：Openweather
